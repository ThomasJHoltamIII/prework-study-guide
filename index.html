CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS is essentially a visual refrence book to designate basic visuals as well as specifiy custom commands and their values

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • To begin commit it starts with ~git add -A, which stages the changes and commit the files
  • WIth the files prepped we can add our notes and start commit ~git commit -m "add your notes"
  • Next step to resync with current version is ~git pull origin main, we should see something like "Up to date"
  • To push up our changes we use ~git push origin "the branch you are pushing"
